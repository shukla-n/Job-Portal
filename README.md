# Job Portal Web Application

A full-stack **Job Portal web application** built using **Django** that connects **job seekers** and **employers** on a single platform.  
The application allows employers to post jobs and manage listings, while job seekers can browse jobs and apply easily.

This project demonstrates practical implementation of **Django MVC architecture**, authentication, database management, and dynamic web pages.
